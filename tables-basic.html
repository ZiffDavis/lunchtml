Tables — Basic

<table>, <tr> & <td>

HTML:

        <table border="1">
            <tr>
                <td>row 1, cell 1</td>
                <td>row 1, cell 2</td>
            </tr>
            <tr>
                <td>row 2, cell 1</td>
                <td>row 2, cell 2</td>
            </tr>
        </table>

RESULT:

row 1, cell 1 row 1, cell 2
row 2, cell 1 row 2, cell 2
